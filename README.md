# Apna.Cut — Video Editing & Post-Production Studio

Welcome to **Apna.Cut**, Pakistan's premier video editing and motion design studio. We craft high-impact narrative edits, motion graphics, colour grading, and sound design for top brands, creators, and filmmakers worldwide.

## Features

- **End-to-End Post Production**: Narrative editing, color grading, sound design, motion graphics.
- **Pakistani & International Clients**: Based in Lahore & Karachi, servicing global campaigns.

## Development

Requires Node.js and npm / bun.

```sh
npm install
npm run dev
```

## Built with

- TanStack Start
- TypeScript
- React
- Tailwind CSS
- Three.js
